# aoju-memory 发布步骤

## 第一步：新建 GitHub Repo

1. 打开浏览器访问：
   👉 https://github.com/new

2. 配置：
   - **Repository name**: `aoju-memory`
   - **Description**: `🐺 Long-term memory + self-evolution skill for OpenClaw AI agents`
   - **Public** ✓（公开）
   - **Add a README file**: ❌（不需要，我们已有）
   - **License**: MIT License ✓

3. 点击 **Create repository**

---

## 第二步：推送代码

在你电脑的终端运行：

```bash
cd /root/aoju-memory
git remote add origin https://github.com/你的GitHub用户名/aoju-memory.git
git branch -M main
git push -u origin main
```

> ⚠️ 如果提示权限不足，先去 https://github.com/settings/tokens 生成一个 token 给 git 用

---

## 第三步：填 Meta 信息

推完后，把你的 **GitHub 用户名** 和 **repo 地址** 发给老二，
我把 `_meta.json` 里的占位符替换掉：

```json
{
  "owner": "你的GitHub用户名",
  "slug": "aoju-memory",
  ...
  "commit": "https://github.com/你的GitHub用户名/aoju-memory/commit/main"
}
```

---

## 第四步：安装试用

发布成功后，用这个命令安装：

```bash
openclaw skills install aoju-memory
```

或手动安装：

```bash
git clone https://github.com/你的用户名/aoju-memory.git
cd aoju-memory
cp -r . ~/.openclaw/workspace/skills/aoju-memory/
```

---

## 安装后验证

```bash
openclaw skills list | grep aoju-memory
python3 ~/.openclaw/workspace/skills/aoju-memory/scripts/mem_status.py
```
